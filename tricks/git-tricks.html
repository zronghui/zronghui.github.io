在GitHub 建立新仓库，注意不要勾选 readme

git init

git remote add origin <你的远程仓库地址>

编写 .gitignore

git add .

git commit -m ‘…’

git push -u origin master

若勾选 readme：

git pull origin master –allow-unrelated-histories

如果进入到merge信息界面，说明成功了，只需要输入:wq,回车，如果不报错误，直接执行下面的语句即可

git push -u origin master
